## Logo Assets

All logo files live in `branding/Logo/` organised by format.

| File | Description |
|------|-------------|
| CMB-Logo | Symbol (icosahedron mark) only |
| CMB-Wordmark | "combinate" text only |
| CMB-Stacked | Symbol above wordmark |
| CMB-Horizontal | Symbol left of wordmark |
| CMB-Logo-spaced | Symbol with extra clear space (PNG only) |
| *-NEG variants | White/reversed versions for dark backgrounds |

**Formats available:** SVG (preferred for digital/web), PNG (transparent background), JPG (white background), EPS (print/press)
